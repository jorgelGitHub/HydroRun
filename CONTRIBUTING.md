# Contribuir a HydroRun

Gracias por mejorar HydroRun. Mantén los textos de interfaz en español y los nombres técnicos de clases, funciones y variables en inglés.

## Flujo sugerido

1. Crea una rama descriptiva.
2. Ejecuta `npm install`.
3. Haz cambios pequeños y verificables.
4. Ejecuta `npm run lint`, `npm test` y `npm run build`.
5. Abre un pull request con resumen y pruebas realizadas.

## Estilo

- Usa TypeScript estricto.
- Evita `any`.
- No guardes información sensible.
- Documenta recursos externos en `ASSETS.md`.
- No uses assets de franquicias comerciales.
