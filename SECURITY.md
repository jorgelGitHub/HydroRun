# Seguridad

HydroRun no usa backend en esta primera versión y guarda únicamente datos mínimos en LocalStorage.

## Reportar problemas

Abre un issue privado o contacta al equipo mantenedor del repositorio.

## Principios

- No incluir claves, tokens ni contraseñas.
- No guardar información sensible.
- Validar datos de formularios.
- Proteger lecturas de LocalStorage frente a valores corruptos.
- Evitar dependencias innecesarias.
